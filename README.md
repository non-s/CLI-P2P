# CLI P2P Chat — Demo Web

Interface interativa de um chat P2P descentralizado via TCP/IP e UDP. Demo no browser.

**[→ Acessar o demo](https://non-s.github.io/Profile-)**

## Sobre o projeto

Chat peer-to-peer sem servidor central. Comunicação direta máquina a máquina usando sockets TCP/IP e UDP.

## Funcionalidades do demo

- Terminal interativo no browser
- Comandos: `/host`, `/connect`, `/nick`, `/ping`, `/peers`
- Simulação de conexão P2P com mensagens de peers
- Histórico de comandos (↑ ↓)

## Stack

HTML · CSS · JavaScript — puro, sem frameworks.
